# Airstream Pressure Washing Inc. - Website

Modern, sleek website for Airstream Pressure Washing Inc. Built with vanilla HTML, CSS, and JavaScript with Framer-inspired design.

## Features

- Modern, minimal design inspired by Framer
- Fully responsive (mobile, tablet, desktop)
- Smooth scroll animations
- Contact form with Netlify Forms integration
- Fast loading - no framework dependencies
- SEO-friendly meta tags

## Deploy to Netlify

### Option 1: Drag and Drop
1. Go to [app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag this entire folder onto the page
3. Your site is live!

### Option 2: Connect GitHub Repository
1. Push this code to a GitHub repository
2. Go to [app.netlify.com](https://app.netlify.com)
3. Click "Add new site" > "Import an existing project"
4. Connect your GitHub account and select this repository
5. Click "Deploy site"

## Form Setup

The contact form is pre-configured for Netlify Forms. It will automatically:
- Detect the form on deploy
- Collect submissions in your Netlify dashboard
- Redirect to `/success` page after submission

To view form submissions:
1. Go to your site in Netlify dashboard
2. Click "Forms" in the sidebar
3. Select the "quote" form

## File Structure

```
├── success.html    # Form success page
├── styles.css      # All styling
├── script.js       # JavaScript interactions

```

## Customization

- **Colors**: Edit CSS variables in `styles.css` under `:root`
- **Content**: Edit text directly in `index.html`
- **Contact Info**: Update in the contact section and footer
